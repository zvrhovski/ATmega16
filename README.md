# Mikroupravljači — ATmega16

Materijali za laboratorijske vježbe kolegija **Mikroupravljači** na Veleučilištu u Bjelovaru, uključujući skeletni kod vježbi, potpuna rješenja i udžbenik.

## Sadržaj

| Modul | Vježbe |
|-------|--------|
| Digitalni izlazi | vjezba411 – vjezba417 |
| Digitalni ulazi | vjezba421 – vjezba425 |
| LCD displej | vjezba511 – vjezba518 |
| EEPROM memorija | vjezba611 |
| ADC | vjezba711 – vjezba715 |
| Timer | vjezba811 – vjezba819 |
| Numerički displej | vjezba911 – vjezba915 |
| USART | vjezba1011 – vjezba1015 |
| Vanjski prekidi | vjezba1111 |
| Senzori i aktuatori | vjezba1211 – vjezba1214 |

Svaki modul sadrži dvije mape:
- **`vjezbe/`** — skeletni kod za izradu vježbi
- **`rjesenja/`** — potpuna rješenja

## Hardver

- Mikroupravljač: **ATmega16**
- IDE: **Atmel Studio**
- Jezik: **C**
